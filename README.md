# Production Grade Large Language Models From Scratch

![](https://automatski.com/wp-content/uploads/2025/05/Automatski-New-Logo.svg)

## About

Large Language Models are fascinating and quite useful. It is immaterial if they will lead to AGI or not, we should use them for what they are worth. It is important to understand how they work and how we can build them from scratch. Hopefully on a standalone laptop.

This Starter Kit has been created by [Automatski](https://automatski.com) and [Wow Internet Labz](https://wowlabz.com). Hopefully it will be useful for learning, teaching and as a reference.

## Installation

Requires Python v3.11+ and Pytorch to run.

Install dependencies:

```sh
pip install tokenizers einops sentencepiece requests
```

Run any llm example using:
```sh
python <filename.py>
```
